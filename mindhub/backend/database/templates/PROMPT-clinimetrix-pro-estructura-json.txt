INSTRUCCIONES PARA CLAUDE:
================================================================================

Cuando recibas información en este formato:

1. **VALIDA** que toda la información esté presente
2. **CONVIERTE** a JSON ÚNICO usando la estructura ClinimetrixPro:
   - metadata (información básica)
   - structure (secciones e ítems)
   - responseGroups (opciones de respuesta compartidas)
   - scoring (método de puntuación y rangos)
   - interpretation (reglas de interpretación)
   - documentation (propiedades psicométricas y referencias completas)

3. **OPTIMIZA** para el motor de renderizado:
   - Agrupa respuestas similares en responseGroups
   - Numera ítems secuencialmente
   - Incluye metadata necesaria para componentes especiales
   - Genera IDs únicos y consistentes
   - Incluye TODA la documentación científica dentro del JSON

4. **VALIDA** el JSON resultante:
   - Estructura completa y válida
   - Todos los ítems tienen respuestas definidas
   - Puntuaciones e interpretaciones son coherentes
   - Documentación científica completa está incluida
   - Instrucciones profesionales y para pacientes incluidas

5. **GUARDA** como archivo .json ÚNICO en el directorio correcto:
   `/Users/alekscon/taskmaster-ai/mindhub/backend/templates/scales/[abreviacion]-[version].json`
